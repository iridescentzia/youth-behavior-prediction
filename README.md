## 🎓 프로젝트 소개

KYPS2018 데이터를 기반으로 청소년 비행행동을 분석하고 예측한 프로젝트입니다.  
Orange3를 활용해 6가지 머신러닝 모델(Logistic Regression, KNN 등)의 성능을 비교했습니다.  
데이터 기반 정책 제안의 의의를 인정받아 교내 공모전 **우수상**을 수상했습니다.
